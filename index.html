Bloodhound

About

Bloodhound is a tool for gathering open source intelligence (OSINT) with a focus on tracking down human trafficking victims and offenders. All tools and links are free and checked regularly to guarantee that Bloodhound stays relevant in the fight against human trafficking.

Bloodhound is an open source project. If you are interested in contributing, visit our GitHub repository.
